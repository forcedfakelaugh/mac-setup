# mac-setup

Automated macOS developer setup script.

## Usage

```bash
git clone https://github.com/forcedfakelaugh/mac-setup.git ~/Developer/mac-setup
cd ~/Developer/mac-setup
bash setup.sh
```

## What it installs

**CLI tools**
`bat` `btop` `btop` `cmatrix` `cowsay` `fd` `gh` `htop` `lazygit` `lolcat` `neovim` `nmap` `nvm` `python3` `ranger` `ripgrep` `tmux`

**Apps**
`raycast` `ghostty` `visual-studio-code` `codex` `firefox` `claude` `chatgpt` `superwhisper` `wispr-flow` `aldente` `tailscale` `vlc` `transmission` `coteditor`

**Fonts**
`JetBrains Mono Nerd Font`

## After running

1. `source ~/.zshrc`
2. Open Raycast → Extensions → Script Commands → Add Directory → `~/raycast-scripts`
